# KUCODE (KU COmmunity of DEvelopers)


# Requirements
python >= 3.6  
Node.js >= 18.0  
docker  
docker-compose  

## plugins in vscode
TypeScript Vue Plugin (Volar)   
Vue Language Features (Volar)  
prettier  
Auto Rename Tag  
Auto Close Tag  
Vetur  

# Running
1. create a .env file on project root and add SECRET_KEY variable (for django settings)
2. Launch docker
3. run command
```
docker compose up
```
